gim_cv.interfaces.tif.rasterio.read_raster_band(path, band=1, target_chunk_size=5120, natural_chunk_multiplier=None)¶

Read a raster band and return a Dask array :param path {string} – path to the raster file:

Keyword Arguments

  • {int} -- number of band to read (default (band) – {1})

  • {int} -- block size multiplier (default (block_size) – {1})

  • -- if target_chunk_size is None, (natural_chunk_multiplier) – fall back on this x natural chunks (along each axis, so mult^2 natural chunks) if this is None, just read natural blocks

gim_cv.interfaces.tif.rasterio.write_raster(path, array, **kwargs)¶

Write a dask array to a raster file If array is 2d, write array on band 1. If array is 3d, write data on each band :param path {string} – path of raster to write: :param array {dask.array.Array} – band array: :param kwargs {dict} – keyword arguments to delegate to rasterio.open:

Examples

# Write a single band raster >> red_band = read_raster_band(“test.tif”, band=1) >> write_raster(“new.tif”, red_band) # Write a multiband raster >> img = read_raster(“test.tif”) >> new_img = process(img) >> write_raster(“new.tif”, new_img)
